# Gap: Can a four-foot, shaded, fishless backyard pond still be worthwhile dragonfly habitat when mosquito control is not the deciding goal?

- **Domain:** Garden wildlife habitat
- **Type:** fit
- **Status:** converging
- **Origin:** GARDEN / pest-control / dragonfly-airforce / predator-recruitment
- **Closure condition:** A human-backed practical design decision that identifies whether the habitat value justifies the constraints of shade, no fish, and limited pond size.

## Sessions

### 2026-07-10 · teacher
- **Scenario / mad-lib:** The user had room for one pond and specified a maximum size, light level, and whether fish were acceptable, then chose whether the project was still worthwhile without proven mosquito-control benefits.
- **User inputs:** Maximum size: four feet. Site: shade. Fish: none.
- **Artifacts:** One design sketch in words tested whether a small shaded fishless pond could be framed as dragonfly habitat rather than as a mosquito-control device.
- **Steer / outcome:** Yes: the user would still build it even if it mainly attracted shade-tolerant dragonflies and did not maximize mosquito control.
- **Mined:** Verified: For this user, the pond remains worthwhile when its purpose is shifted from pest control to wildlife habitat. The key design tradeoff is accepted: shade may reduce dragonfly diversity, but that does not invalidate the project because mosquito reduction is no longer the success criterion.
- **Left open:** Which shade-tolerant dragonfly or damselfly species are realistic for the user's region, and what exact plant and depth mix best supports them without creating persistent mosquito breeding.
